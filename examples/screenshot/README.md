# Screenshot Generator

Test app for a [Screenshot Generator](https://toga.beeware.org).

## Quickstart

To run this example:

Set up a development environment:

```
$ python -m venv venv
$ ./venv/bin/activate
(venv) $ python -m pip install briefcase
```

To run in development mode:

```
(venv) $ briefcase dev
```

To run as a packaged app:

```
(venv) $ briefcase run
```
