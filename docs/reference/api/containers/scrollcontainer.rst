ScrollContainer
===============

A container that can display a layout larger than the area of the container, with
overflow controlled by scroll bars.

   :align: center

Usage
-----

.. code-block:: python

    import toga

    content = toga.Box(children=[...])

    container = toga.ScrollContainer(content=content)
