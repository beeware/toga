Number Input
============

The Number input is a text input box that is limited to numeric input.

.. figure:: /reference/images/NumberInput.jpeg
    :align: center

Usage
-----

.. code-block:: Python

    import toga
    
    textbox = toga.NumberInput(min_value=1, max_value=10)
